In `fallout2.cfg`, find and enable the following settings:

```ini
[debug]
mode=environment
output_map_data_info=1
show_load_info=1
show_script_messages=1
show_tile_num=1

[sound]
debug=1
debug_sfxc=1
```

In `ddraw.ini`:

```ini
[Debugging]
Enable=1
DebugMode=2
Init=1
Hook=1
Script=1
Criticals=1

```
